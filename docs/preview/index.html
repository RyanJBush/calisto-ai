Callisto: Document RAG Portfolio Demo

Built by a University of Maryland student studying Information Science and Electrical Engineering with a Business minor. Callisto demonstrates practical document indexing, hybrid retrieval, weighted reranking, and heuristic answer synthesis with source citations.

What the system does

Indexes documents

Uploads text content, chunks it, and stores retrieval-ready records.

Runs hybrid search

Combines lexical and vector retrieval to gather candidate chunks.

Applies weighted reranking

Reorders results with feature-weight scoring for clearer relevance.

Assembles answers with citations

Uses template-based synthesis and returns supporting snippets.

How it works (30-second view)

  1. Upload a document in the UI and store chunked content.
  2. Create deterministic hash-based embeddings plus lexical indexes.
  3. Retrieve candidates and apply weighted reranking.
  4. Return a template-based answer with supporting citations.
