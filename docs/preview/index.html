Callisto: Enterprise RAG Knowledge Platform

A demo-scale full-stack RAG project that ingests documents, retrieves relevant context, applies weighted reranking, and assembles citation-grounded answers with transparent backend architecture.

Key capabilities

Document ingestion

Upload and process files into chunked records for retrieval.

Hybrid retrieval

Combines lexical and vector signals to gather relevant context.

Weighted reranking

Reorders candidates using configurable feature blending.

Grounded response assembly

Returns heuristic answer synthesis with source-aware citations.

Tech stack

FastAPIReact + TypeScriptPostgreSQLSQLAlchemyFAISSDockerJWT Auth + RBAC

Architecture (high-level)

How it works

  1. Ingest documents and split them into indexed chunks.
  2. Create deterministic hash-based embeddings and lexical indexes.
  3. Retrieve candidates from hybrid search and apply weighted reranking.
  4. Assemble a response using template-based heuristic synthesis with citations.
